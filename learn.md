# LEARN

### using storybook 

  1. add 
  ```
  yarn add -dev @storybook/react
  ```
  2. package.json file 추가
  ```
  ... 
  "scripts": {
    "name": start-storybook -p <port number> -c <dir>

    // ex) "storybook": "start-storybook -p 9001 -c .storybook"
  }
  ```
  npm run start storybook

  3. config.js 와 story 연결
